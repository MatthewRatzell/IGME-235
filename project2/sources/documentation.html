Documentation

A large majority of this project was spent working on the Javascript and learning how to access and manipulate the GIPHY API, once I had created a somewhat functional setup I than refactored most of my code by implementing helper functions where applicable to make my coding more responsive. After I had all that worked out I than attempted to make the site a little more pretty and appealing, and I think I did so(hope so) using decent design choices

Resources Used

  • GIPHY API with a GIPHY beta key
  • A large amount of w3 school resources(w3 schools)
  • This video really helped me understand how to better utilize the GIPHY API and API's in general(Steve Griffith's GIPHY Tutorial)

Known Issues

Unfortuetly the site does have a few issues I was unable to work out, these are listed below

  • Sometimes when the site is first loaded up the user has too change the rating before searching will display any GIFS, this is strange considering I have it set the max rating to 'R' at startup(this may be fixed as of now and I believe it is, but I figure I should leave this here in case you run into the problem as well)
  • If the user favorites an image and than finds the image again under a NEW max rating the image wont be recognized and the user will be able to favorite the same image again, this does not apply to images found under the same max rating, the code will recognize that the image does exist in the favorites list and will appear as already favroited
